Menurut Dennis, (2012) terdapat 4 jeis kebutuhan nonfungsional utama yang dapat menjadi penting dalam merancang arsitektur, salah satunya adalah kebutuhan operasional.

Kebutuhan operasional berarti menentukan lingkungan operasi pada bagian mana sistem harus bekerja dan bagaimana sebuah sistem dapat berubah seiring waktu. Lingkungan operasional biasanya terdiri dari sistem operasi, perangkat lunak sistem, dan sistem informasi yang berinteraksi dengan sistem ataupun lingkungan fisik jika ada yang penting untuk aplikasi

Pada konteks perusahaan pengiriman / ekspedisi barang, kebutuhan operasional secara umum mencakup:
1. Lingkungan teknis sistem operasi dan perangkat lunak
   Dikarenakan sistem ekspedisi beroperasi dalam ekosistem yang kompleks, maka dibutuhkan:
   - OS Server yang stabil seperti Linux atau Windows Server
   - DBMS yang andal seperti PostgreSQL, MySQL, atau Oracle
   - Platfrom untuk integrasi dengan layanan lain
   - Sistem untuk pelacakan, manajemen gudang dan transportasi
2. Kebutuhan perangkat keras
   Dalam melakukan operasional, ekspedisi memerlukan perangkat fisik yang memadai untuk mendukung proses logistik, seperti:
   - Server dan penyimpanan data, baik lokal maupun cloud.
   - Perangkat mobile untuk kurir, seperti smartphone atau handheld scanner untuk input status pengiriman.
   - Printer label barcode, timbangan digital, dan RFID reader di gudang.
   - Perangkat jaringan stabil (router, access point, load balancer).
3. Integrasi dengan Sistem Eksternal
    Perusahaan ekspedisi biasanya harus berinteraksi dengan banyak pihak. Maka diperlukan:
   - Integrasi dengan marketplace (Tokopedia, Shopee, dll).
   - Integrasi dengan sistem pembayaran, faktur, dan sistem pelacakan nasional/internasional.
   - Penyedia layanan API untuk mitra atau agen logistik.
4. Keamanan Operasional
   Karena data yang dikelola sensitif, sistem harus mendukung:
   - Enkripsi data pengiriman dan identitas pelanggan.
   - Kontrol akses berbasis peran untuk staf gudang, admin, dan kurir.
   - Audit log untuk semua aktivitas penting.
   - Proteksi terhadap serangan seperti DDoS, brute force, dan data breach.

###### Kesimpulan
Kebutuhan operasional pada perusahaan ekspedisi berfokus pada memastikan bahwa sistem dapat berjalan secara andal di berbagai lingkungan teknis dan fisik, mampu berintegrasi dengan banyak pihak, mendukung volume transaksi besar, dan aman digunakan dalam operasional pengiriman barang sehari-hari.
