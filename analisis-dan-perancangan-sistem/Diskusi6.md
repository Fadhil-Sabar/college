Object diagram merupakan contoh nyata dari sebuah Class Diagram. Dimana class diagram hanya berupa struktur awal dari sebuah sistem. 

Saya akan menggunakan contoh flow sederhana dari aplikasi SITTA (Sistem Informasi Tiras dan Transaksi Bahan Ajar) namun hanya pada fitur untuk tracking bahan ajar.

![alt text](Diskusi6Alper.drawio(1).png)

*Diagram ini saya gambar menggunakan aplikasi drawio*.

Pada alur aplikasi SITTA (Suffandi, U. U. et al 2021), setelah user melakukan login, maka dapat melakukan tracking bahan ajar dengan melakukan input nomor tracking. Maka akan mendapatkan informasi data tracking yaitu berupa trackingId, nomorResi, namaPemesan, alamat, jenisEkspedisi, tanggalPengiriman, totalPembayaran.

Kemudian didapatkan juga informasi tentang pengiriman, terdapat informasi status dan kurir beserta riwayat dari pengiriman tersebut.

Sehingga ini merupakan keseluruhan flow dari tracking bahan ajar dalam bentuk object diagram
